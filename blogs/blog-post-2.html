Subgame-Perfect Equilibrium in the Sequential Box-Stealing Game

Game Definition

There are \(N\) players indexed \(1,2,\dots,N\).

Each player \(i\) initially owns a box containing a random variable

\[X_i \sim \text{Uniform}(0,1)\]

with all draws independent.

Player 1 begins by opening their box, revealing \(X_1\).

For each player \(i = 2,\dots,N\), the player chooses one action:

  • Open: Open their own box, revealing \(X_i\), and keep it.

  • Steal: Take a revealed value from another player. The stolen player receives player \(i\)’s box and must immediately open it.

The game ends after player \(N\) acts.

Each player’s payoff equals the value of the box they hold at the end.

Players maximize expected payoff and all revealed values are common knowledge.

Last Player Strategy

Let \(M\) denote the maximum revealed value before the last player moves.

Theorem 1. The last player steals the maximum revealed value if and only if \[M \ge \frac{1}{2}.\] Otherwise the player opens their own box.

Proof. If the player steals, they can obtain value \(M\) for certain.

If the player opens their own box,

\[\mathbb{E}[X_N] = \frac12.\]

Thus stealing is optimal whenever \(M \ge 1/2\). ◻

Second-to-Last Player

Let the revealed values before player \(N-1\) moves be

\[v_1 \ge v_2 \ge \dots \ge v_k\]

with

\[M = v_1, \qquad S = v_2\]

when \(k \ge 2\).

Secure Steal

Lemma 1. If player \(N-1\) steals a non-maximum value \(y < M\), the last player will not steal from them.

Proof. The maximum value after the move will be \(\max(M,X_{N-1})\), which is strictly greater than \(y\). The last player therefore steals the maximum instead of \(y\). ◻

Thus stealing \(S\) yields payoff

\[\text{Payoff} = S\]

with certainty.

Risky Maximum Steal

If player \(N-1\) steals \(M\), the expected payoff becomes

\[E = (1-M)M + M\cdot \frac12\]

because:

  • with probability \(1-M\), the new box value exceeds \(M\), redirecting the final steal away

  • with probability \(M\), the maximum remains \(M\) and is stolen

Thus

\[E = \frac32 M - M^2.\]

Opening

If the player opens their own box and \(M \ge 1/2\),

\[E = \frac{M^2 + 1 - M}{2}.\]

If \(M < 1/2\), the expected value simplifies to

\[E = \frac{3}{8}.\]

Optimal Strategy

Proposition 1. For the second-to-last player:

  • If \(M < 1/2\)

    \[\text{Steal } M \quad \text{iff} \quad M > 3/8\]

    otherwise open.

  • If \(M \ge 1/2\)

    • if \(k=1\), steal \(M\)

    • if \(k\ge2\), choose between

      \[\text{secure payoff } S\]

      and

      \[\text{risky payoff } \frac32 M - M^2\]

      and take the larger.

Example

Suppose the revealed values include

\[0.99 \quad \text{and} \quad 0.98.\]

The risky steal value becomes

\[T(0.99) = \frac32(0.99) - 0.99^2 = 0.5049.\]

Since

\[0.98 > 0.5049\]

the second-to-last player prefers stealing \(0.98\).

The last player will then steal \(0.99\) from someone else.

Thus the equilibrium strategy is to hide beneath the maximum.

General Backward Induction

Let \(r\) denote the number of players remaining after the current player.

There exists a threshold \(c_r\) such that in canonical subgames the player accepts a revealed value \(m\) whenever

\[m \ge c_r.\]

The thresholds satisfy

\[c_0 = \frac12, \qquad c_1 = \frac38\]

and converge rapidly to

\[c_\infty \approx 0.30685.\]

For large games such as \(N=101\), nearly all early players behave as if the threshold were approximately

\[0.307.\]

Conclusion

The sequential box-stealing game exhibits a structured equilibrium governed by backward induction.

Key insights include:

  • The last player compares the maximum revealed value to the expected value of a fresh draw.

  • The second-to-last player may strategically avoid stealing the maximum to prevent becoming the final target.

  • Early players behave according to a nearly stationary threshold rule.

The resulting equilibrium highlights how strategic exposure to future theft shapes optimal decisions.
